# Event Management
Continuing in the event page, where it should have also
1. Table with all the participants, displaying
  a. Name, Type, Status, Actions buttons (check in and check out)
2. Check in rules
  a. A VIP participant can check-in and check-out how many times it wants
  b. A normal participant can check-in and check-ou only once
  c. A closed event should not allow any check-in
3. Every action should display a toast notification in the bottom-right of the screen.
  a. Check-in success (VIP): "Entry registered successfully"
  b. Check-in success (Normal): "Check-in registered. No further check-ins allowed"
  c. Check-in error (Normal, 2nd attempt): "Participant has already checked in"
  d. Check-in blocked (closed event): "Event is closed. No check-ins allowed"
  e. Check-out success (VIP): "Exit registered. Participant may re-enter"

Toast should auto-dismiss after 4 seconds. 
Errors should use a red accent, success a green accent.
